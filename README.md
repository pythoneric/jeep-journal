# BitEric Jeep

Personal journal for Jeep Wrangler and Gladiator vehicles. Tracks maintenance, fuel, and modifications offline.

## Installation

1. `npm install`
2. `npm start`
3. Open http://localhost:8080/jeep.html

## Features

- Offline-first PWA
- Vehicle profiles
- Maintenance log with cost tracking
- Fuel log with MPG calculation and charts
- Modifications tracking
- Bilingual (EN/ES)
- Dark/light theme
- Data export/import

## Icons

Replace `icon-192.png` and `icon-512.png` with your own 192x192 and 512x512 PNG icons before installing the PWA.

# Diario BitEric Jeep

Diario personal para vehículos Jeep Wrangler y Gladiator. Rastrea mantenimiento, combustible y modificaciones sin conexión.

## Instalación

1. `npm install`
2. `npm start`
3. Abre http://localhost:8080/jeep.html

## Características

- PWA primero sin conexión
- Perfiles de vehículos
- Registro de mantenimiento con seguimiento de costos
- Registro de combustible con cálculo de MPG y gráficos
- Seguimiento de modificaciones
- Bilingüe (EN/ES)
- Tema oscuro/claro
- Exportación/importación de datos

## Iconos

Reemplaza `icon-192.png` e `icon-512.png` con tus propios iconos PNG de 192x192 y 512x512 antes de instalar la PWA.
